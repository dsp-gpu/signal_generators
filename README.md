# DspSignalGenerators

> Part of [dsp-gpu](https://github.com/dsp-gpu) organization.

Signal generation library: CW, LFM, Noise, Script (DSL), FormSignal, DelayedFormSignal.

## Dependencies
- DspCore, DspSpectrum (lch_farrow for DelayedFormSignal)
- ROCm: hip, hiprtc (ScriptGenerator runtime compilation)
